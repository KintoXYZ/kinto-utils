# Kinto Utils (DRAFT)

## Install

```
git clone https://github.com/KintoXYZ/kinto-utils.git

cd kinto-utils

yarn install
```

## Running signer

`npx ts-node ./src/sign.ts <privateKey> <chainId>`

## Deploy Counter contract

`npx ts-node src/samples/deployContract.ts`

## Incremenet counter

`npx ts-node src/samples/callContract.ts`
